Welcome to QuickClaw

One-time setup takes about 2 minutes.
After that, everything works with a double-click.

1

Open Terminal

Terminal is a built-in app on every Mac. The fastest way to open it:

⌘ Cmd + Space
to open Spotlight, then type Terminal and press Enter
Don't see it?

You can also find Terminal in Applications → Utilities → Terminal.

2

Paste this command

Click the Copy button below, go to Terminal, and paste it in. Then press Enter.

$ Detecting folder...

Detecting the correct folder path from this file...

Command not detected? Click here
If the command above still says Detecting folder..., your browser likely blocked local path detection.

Use this format manually:

bash /full/path/to/your/QuickClaw-folder/QuickClaw_Setup.command

Fast method: Open Terminal, type bash (with a space), then drag QuickClaw_Setup.command from Finder into Terminal and press Enter.
3

Follow the prompts

The setup will ask a couple of yes/no questions and show progress. It looks something like this:

╔════════════════════════════════════════════╗ ║ Welcome to QuickClaw ║ ╚════════════════════════════════════════════╝ This will: 1. Allow all QuickClaw scripts to run on your Mac 2. Install OpenClaw and its dependencies 3. Set up the QuickClaw dashboard Ready to start? (Y/n) Y ✓ QuickClaw_Install.command ✓ QuickClaw_Launch.command ✓ QuickClaw_Stop.command ✓ QuickClaw_Verify.command ✓ dashboard-files/ Done. 5 scripts unlocked.

Just say yes when it asks and let it run. It usually takes 1–2 minutes.

✓

You're all set

That one-time setup is done. From now on, just double-click to use QuickClaw:

QuickClaw_Launch.command Start OpenClaw + dashboard
QuickClaw_Stop.command Stop everything
QuickClaw_Verify.command Check that everything's healthy
QuickClaw Doctor.command Troubleshoot problems

Once launched, open http://localhost:3000 in your browser for the dashboard.
