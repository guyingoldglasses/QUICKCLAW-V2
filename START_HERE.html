Telegram Bot

Chat with your OpenClaw instance from Telegram
‚ñ∏

What this does: Creates a Telegram bot that talks to your OpenClaw instance, so you can chat with it from your phone or desktop.

Create your bot:

1
Open Telegram and search for @BotFather (the official Telegram tool for making bots)
2
Send /newbot to BotFather
3
BotFather will ask you for a display name (can be anything, like "My OpenClaw Bot")
4
Then it asks for a username ‚Äî this must end in "bot" (like myclaw_bot). Has to be unique across Telegram.
5
BotFather gives you a bot token ‚Äî a long string like 7123456789:AAF.... Copy this. You'll need it next.

Connect it to OpenClaw:

6
Open your OpenClaw config file at openclaw/config/default.yaml
7
Add a Telegram section with your bot token:

telegram:
  bot_token: YOUR_TOKEN_HERE
8
Save the file, then restart OpenClaw using QuickClaw_Stop.command followed by QuickClaw_Launch.command
9
Open Telegram, find your bot by its username, and send it a message. It should respond through OpenClaw.
Tip: You can also use BotFather to customize your bot's profile picture and description. Send /mybots to BotFather to manage your existing bots.

Claude (Anthropic) API Key

Use Claude models as your AI provider
‚ñ∏

What this does: Lets OpenClaw use Anthropic's Claude models (like Claude Sonnet or Opus) for conversations.

Get your API key:

1
Go to console.anthropic.com
2
Sign up for an account if you don't have one (email + phone verification)
3
Once logged in, click API Keys in the left sidebar
4
Click Create Key, give it a name (like "QuickClaw"), and copy the key. It starts with sk-ant-...
5
You'll need to add billing ‚Äî go to Plans & Billing in the sidebar and add a payment method. API usage is pay-as-you-go.

Add it to OpenClaw:

6
Open openclaw/config/default.yaml and add:

anthropic:
  api_key: sk-ant-YOUR_KEY_HERE
7
Restart OpenClaw (Stop, then Launch)
Tip: Copy the key immediately when it's shown ‚Äî Anthropic only shows it once. If you lose it, just create a new one.

OpenAI API Key

Use GPT models as your AI provider
‚ñ∏

What this does: Lets OpenClaw use OpenAI's GPT models (like GPT-4o) for conversations.

Get your API key:

1
Go to platform.openai.com
2
Sign up or log in with your OpenAI account (same account as ChatGPT, if you have one)
3
Click the gear icon (Settings) in the top right, then go to API Keys
4
Click Create new secret key, name it, and copy it. It starts with sk-...
5
Add billing if needed ‚Äî go to Settings ‚Üí Billing and add a payment method

Add it to OpenClaw:

6
Open openclaw/config/default.yaml and add:

openai:
  api_key: sk-YOUR_KEY_HERE
7
Restart OpenClaw (Stop, then Launch)
Tip: Like Anthropic, OpenAI only shows the key once. Copy it right away. If you lose it, delete the old one and make a new one.

OpenAI OAuth (Zero-Auth)

Connect to OpenAI without managing API keys
‚ñ∏

What this does: Instead of creating and pasting an API key, OAuth lets you sign in to OpenAI directly and authorize OpenClaw to use your account. No keys to copy or manage.

How to set it up:

1
Open the QuickClaw dashboard at localhost:3000
2
Scroll to the Add-ons & Integrations section
3
Find the OpenAI card and look for an Authorize with OpenAI or Sign in with OpenAI option
4
Click it ‚Äî you'll be redirected to OpenAI's website to sign in and approve the connection
5
After approving, you'll be sent back to the dashboard. OpenClaw now has access ‚Äî no API key needed.

If OAuth isn't available yet:

OAuth support depends on your OpenClaw version and configuration. If you don't see the sign-in option, use the standard API key method above instead. You can check for updates to OpenClaw which may add OAuth support for additional providers.

Tip: OAuth is the easier option when it's available ‚Äî nothing to copy, nothing to lose. It uses your existing OpenAI account and billing. You can revoke access anytime from your OpenAI account settings under "Authorized applications."
